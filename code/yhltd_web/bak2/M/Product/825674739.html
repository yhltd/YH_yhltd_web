人事系统方便快捷管理企业人员

更新：2018-6-2 1:47:24 点击：
  • 品牌： 人事系统
  • 型号： YHLTD010
产品介绍

　　 系统会提供各种功能来满足企业对人力资源自动化管理的即时使用需求，例如电子招聘、员工信息记录、出勤和资源使用管理、薪酬和工资单、福利计划、员工绩效、人才盘点、在线假期管理以及员工自助功能。这些功能让企业更有效地管理和控制人力资源。同时，人事管理的自动化，也大大地提高了人力资源管理部门的工作效率。

　　


　　云合未来定制人事系统

　　人事档案中包括薪酬记录、考勤记录、绩效记录、培训记录、社保记录、调岗记录、调薪记录、奖惩记录等常用数据子集。用户也可自行增加新的数据子集。可以针对子集进行独立的导入、导出、统计分析。
