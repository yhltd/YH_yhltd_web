云合人工智能系统定制款网爬系统

更新：2018-5-23 15:03:46 点击：

  • 产品型号 yhwp
产品介绍

利用该系统，用户可以有选择的采集网络中的各种信息资源，避免频繁的网络登录操作和枯燥的手工信息采集过滤工作，更快捷、更高效、更准确的收集有用的资源信息。根据自己的需要，抓取指定网站的资源信息内容，过滤掉所有的广告以及与资源信息无关的内容。可以根据根据自己的需要，指定什么时间抓取资源信息。开机自动运行，系统丢电或者意外重新启动，不需要登录系统，也可以根据指定的时间抓取资源信息自动更新程序和标签，客户端可以根据当前的程序版本和标签内容，自动检测有没有新的版本和标签，实现自动升级，不需要自己维护。可以对抓取的资源信息，进行浏览，可以按照类别来浏览，也可以根据自己的兴趣自定义类别浏览。同时支持对感兴趣的内容进行浏览的功能。分应用程序和WEB版本，WEB版本同时支持对抓取的资源信息的编辑功能。应用程序浏览可以提示栏的形式显示在桌面上。
