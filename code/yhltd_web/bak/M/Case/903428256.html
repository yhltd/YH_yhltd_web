知识图谱无疑能为人们带来很

详细介绍

　　Google此前的搜索方式以文字为主，默认每页显示十个搜索条目，虽然含有链接、图片及页面预览等元素，但仍然需要用户仔细阅读页面以判断。而新上线的知识图谱则有了更丰富直观的内容，它集成于搜索页的右栏，可为用户提供地图、维基百科条目等各种各样的知识。

　　知识图谱无疑能为人们带来很大的便利。比如说，用户搜索“taj mahal”(泰姬陵)的时候，Google图谱会为你显示泰姬陵的地图位置、维基百科介绍、建筑相关知识(高度、开放时间、设计师等)，还会给出和taj mahal同名的人物及赌场等信息，对比原先以文字为主的搜索结果呈现方式，极大程度地提升了用户体验。
